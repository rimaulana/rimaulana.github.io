Menambahkan Interface Baru Pada Ubuntu Server

Read in English: Enabling New Interface in Ubuntu Server
2012-01-07
networking konfigurasi linux
145 kata waktu baca 1 menit
Ceritanya kemarin malam aku ganti salah satu network interface pada Ubuntu Server-ku (Ubuntu 11.04 Natty Narwhal). Niatnya adalah upgrade dari Fast Ethernet jadi Gigabit Ethernet. Setelah aku ganti, aku turned on komputerku. Alhasil indikator LED pada interface yang baru mati (Normalnya akan nyala saat komputer sedang on dan interface terhubung ke jaringan). Untuk meyakinkan 100% bahwa interface baru tersebut tidak bekerja. Aku jalankan perintah user@ubuntu:~$ ifconfig dan hasilnya sesuai dengan prediksi saya.
