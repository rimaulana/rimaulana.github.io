Instalasi LAMPP Pada Ubuntu Server

2012-02-08
konfigurasi linux
166 kata waktu baca 1 menit

Tabel Isi

Ini pengalaman pribadi saya saat menginstal Ubuntu Server pada komputer saya. Komputer tersebut ingin saya jadikan sebagai web server, tentu saja saya akan butuh untuk menginstal OpenSSH dan LAMPP. Ubuntu Server pada dasarnya menyediakan pilihan software yang ingin kita instal pada komputer kita saat kita melakukan instalasi. Yakni pada saat kita sampai pada halaman software selection.

Sialnya. Pada saat itu saya tidak tahu bagaimana cara memilih aplikasi yang ingin saya instal. Akhirnya tanpa pikir panjang mungkin tombol yang digunakan itu adalah ENTER. Saya tekan dan alhasil step tersebut berhasil saya lewati tanpa berhasil memilih baik OpenSSH atau LAMPP untuk diinstall.

Usut punya usut ternyata hal tersebut masih bisa kita lakukan setelah instalasi OS selesai. yakni dengan perintah tasksel.

user@ubuntu:~$ sudo tasksel

Setelah anda eksekusi perintah tersebut akan muncul window yang sama persis saat anda melakukan instalasi. Ternyata tombol yang digunakan untuk memilih software yang kita ingin install adalah SPACE.

Pilih software yang anda inginkan kemudian tekan ENTER dan jawab semua pertanyaan yang akan diajukan dengan jujur.
