Postingan kali ini adalah lanjutan dari postingan IDN Networking Competition 2017: Mikrotik Bridging Challenge Solution, pastikan kamu telah membaca postingan tersebut, kalau tidak saya jamin pasti akan sedikit bingung. Simulasi yang akan saya paparkan menggunakan teknologi sebagai berikut:

  • GNS3 versi 1.5.2 on Windows (64-bit) with Python 3.5.1 Qt 5.6.0
  • Qemu-2.4.0 running on Local Windows (64-bit)
  • Mikrotik CHR 6.40.1

Topologi yang saya gunakan adalah sebagai berikut:

Pertama yang akan kita lakukan adalah verifikasi dari konektifitas dari tiap lantai (table of content ada di samping kanan)

Lantai 1

Pada lantai 1, test yang akan kita lakukan adalah:

No Check List Tujuan
1 Ping dari R-L1 ke Core-JKT Verifikasi konektifitas IP dari R-L1 ke Core-JKT
2 PC1 IP DHCP request Verikasi apakah DHCP server untuk VLAN 10 di Core-JKT berjalan dan dapat terdistribusi di R-L1
3 PC1 ping ke gateway Verifikasi konektifitas IP ke Core-JKT
4 PC2 IP DHCP request Verikasi apakah DHCP server untuk VLAN 20 di Core-JKT berjalan dan dapat terdistribusi di R-L1
5 PC2 ping ke gateway Verifikasi konektifitas IP ke Core-JKT
6 PC3 IP DHCP request Verikasi apakah DHCP server untuk VLAN 30 di Core-JKT berjalan dan dapat terdistribusi di R-L1
7 PC3 ping ke gateway Verifikasi konektifitas IP ke Core-JKT
8 Traceroute dari R-L1 ke PC1 Verifikasi bahwa default gateway R-L1 bekerja dengan baik untuk koneksi internet

Traceroute dari R-L1 ke PC1

Karena pada tahap ini koneksi ke internet belum terbangun, untuk membuktikan bahwa apabila Core-JKT sudah terkonfigurasi internet maka R-L1 dapat pula terhubung ke internet, kita akan mecoba mengkontak alamat IP diluar subnet management R-L1 untuk membuktikan bahwa default route R-L1 berjalan.

[admin@R-L1] > /tool traceroute 192.168.10.254
# ADDRESS                          LOSS SENT    LAST     AVG    BEST   WORST STD-DEV STATUS
1 192.168.99.1                       0%   17   2.2ms     2.2     1.7     5.1     0.8
2 192.168.10.254                     0%   17   4.1ms       8     2.9    69.7    15.4
