Instalasi XAMPP pada Sistem Operasi Linux

2011-07-03
configuration indonesia linux
256 words 2 min read

Table of Contents

XAMPP, kependekan dari X(Linux atau Windows atau Mac OS X atau Solaris), Apache, MySQL, PHP, Perl, merupakan aplikasi yang terdiri dari webserver (Apache), database server (MySQL) serta FTP server (ProFTPD) yang diperuntukkan untuk lingkungan development. Seperti yang disebutkan diatas XAMPP dapat berjalan pada sistem operasi baik berbasis WINDOWS maupun UNIX.

Pada kesempatan kali ini saya akan menjelaskan cara instalasi XAMPP pada sistem operasi linux. Berikut beberapa spesifikasi yang digunakan.

  • Sistem Operasi Ubuntu 11.04 (Natty Narwhal)
  • XAMPP versi 1.7.4

Berikut langkah-langkah instalasi XAMPP pada sistem operasi Linux.

Langkah 1: Download installer XAMPP untuk Linux.

Langkah 2: Masuk ke terminal dan ubah direktori ke tempat anda menyimpan file yang baru saja di download. Dalam kasus ini file saya terdapat di folder Downloads.

user@ubuntu:~$ cd Downloads/

Langkah 3: Ubah ke mode super user dengan mengunakan perintah sebagai berikut.

user@ubuntu:~$ sudo su

Langkah 4: Extract file yang barusan di download ke folder /opt dengan perintah sebagai berikut.

user@ubuntu:~$ tar xvfz xampp-linux-[x].tar.gz -C /opt
note

Pada xampp-linux-[x].tar.gz, tanda [x] merupakan versi dan versi yang saya gunakan adalah 1.7.4

Langkah 5: Jalankan service dengan perintah berikut.

user@ubuntu:~$ /opt/lampp/lampp start

Langkah 6: Verifikasi output, jika tidak terdapat masalah dalam instalasi maka pada terminal akan keluar output sebagai berikut.

Starting XAMPP for Linux 1.7.4...
XAMPP: Starting Apache with SSL (and PHP5)...
XAMPP: Starting MySQL...
XAMPP: Starting ProFTPD...
XAMPP for Linux started.

Demikian cara instalasi XAMPP pada sistem operasi linux. Mudah bukan? dan semoga dapat membantu. Terima Kasih

Referensi :
